# Sprint4_Yanch

Этот проект представляет автоматизированные UI-тесты, написанные на Java с использованием Selenium WebDriver и Maven. Цель — автоматизация проверки веб-интерфейса и функциональности сервиса заказа самоката

## Технологии

| Название                  | Версия   |
|---------------------------|----------|
| Java                      | 11       |
| Maven                     | 3.9.0    |
| JUnit                     | 4.13.2   |
| Selenium WebDriver        | 4.33.0   |
| WebDriverManager          | 6.3.0    |
| Logback                   | 1.2.11   |
| Аннотации (JetBrains)     | RELEASE  |

## Настройка среды

Клонировать репозиторий: https://github.com/yanchilenkoaa-cmd/Sprint4_Yanch Импортировать проект в IntelliJ IDEA. Убедиться, что установлен JDK 11 Maven подтянет зависимости автоматически при открытии проекта

## Запуск тестов

mvn clean test
